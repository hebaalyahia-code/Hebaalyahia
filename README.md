# Heba Signature | هبة سيجنتشر

A professional, minimalist, and luxury-aesthetic portfolio website built using vanilla HTML and CSS. The design focuses on a clean layout, modern typography, and bilingual support (Arabic and English), aiming to leave a lasting impression of elegance and high quality.

## Features ✨
- **Bilingual Support:** Fully implemented pages for both Arabic (RTL) and English (LTR) audiences.
- **Minimalist & Luxury Aesthetic:** Elegant color scheme, smooth transitions, and high-end typography (Cairo/Harir/Gurmukhi MN).
- **Responsive Design:** Optimized for seamless viewing on desktop, tablet, and mobile devices.
- **Vanilla Tech Stack:** Pure HTML5 and CSS3 without the overhead of heavy frameworks or JavaScript.

## File Structure 📂
- `index.html` - Main Arabic portfolio page.
- `index-en.html` - English version of the portfolio.
- `style.css` - Global stylesheet with custom variables and responsive design rules.
- `logo.svg` - Official branding logo.

## How to Run 🚀
Simply open `index.html` (for Arabic) or `index-en.html` (for English) in any modern web browser. No local server setup is required.

## Technologies Used 🛠️
- HTML5
- CSS3 (Custom Variables, Flexbox, CSS Grid)
